# Business-Performance-Dashboard
An Interactive Excel dashboard with KPI's, Slicers, Timeline, and Advanced Charts for Business Insights.

I have analyzed the revenue, returns, dicounts, and Some KPI's like Total Revenue, Net Revenue, Average Order Value, etc.

I have worked with Power Pivot to create KPI's and Calculated Columns, Pivot Table, GETPIVOTDATA formula, and Slicers/Timeline.

## Features
- 💰 KPI cards (Revenue, Net Revenue, Return Rate, Discounts, Delivery Days)
- 🌍 Region-wise analysis (Revenue & Return Rate)
- 🛒 Product Tree Map
- 📅 Timeline slicers (Year, Quarter, Month)
- 🎟 Promotion Code vs Return Rate chart
- 💳 Payment Method share
- 👩‍💼 Salesperson performance

## Tools Used
- Microsoft Excel
- Power Pivot
- Pivot Tables & Charts
- GETPIVOTDATA formulas
- Slicers for interactivity

## How to Use
1. Download the Excel file.
2. Open in Microsoft Excel.
3. Use slicers (Date, Region, Product, Promotion Code) to interact with the dashboard.
